# About app

## Description

`kafijas-pupina-rn-expo` is cross-platform mobile app for shopping coffee and tea from local company.

App's vision: fast and easy way to order coffee and tea online.

## Main features

- creating an order online
- order's basket
- configuring delivery
- payment processing
- orders history
- my profile with delivery information

> Note: this is work-in-progress project, some features has not been implemented yet.

## Technology stack

Written in React Native. I use Expo to simplify app testing and integration of basic libraries and services. Maybe will consider ejecting from Expo in future.

This is not vanilla JS code, I use TypeScript for type-safety.
